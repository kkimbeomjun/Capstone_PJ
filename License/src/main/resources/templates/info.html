사용자 정보 확인하기

사용자정보 창은 현재 사용자가 접근 가능한 정보를 제공합니다.
여기에는 이메일, 제품 키, 라이선스 만료일, 그리고 라이선스 등록일이 포함됩니다..
아래 부분에는 사용자가 보유한 엑셀 파일을 직접 관리하는 테이블들의 이름이 표시됩니다.
테이블의 이름을 확인한 후, 각 테이블의 라이선스를 수정, 삭제, 추가 등 다양한 기능을
활용하여 직접 라이선스를 효과적으로 관리할 수 있습니다.

  • 3
    삭제 버튼을 누르고 데이터를 수정합니다

※ 주의사항

테이블의 관리하기전에 테이블을 보고 위의 사진과 같이 빈칸이 있으면
빈칸을 누르시고 데이터를 삭제해서 빈칸을 삭제해주세요.
해당 빈칸을 삭제하지 않고 진행할시에 데이터 관리에 오류가 발생할 수 있습니다.
